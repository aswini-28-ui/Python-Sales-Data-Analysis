# 📊 Python Sales Data Analysis

## Project Overview
This project analyzes a sample sales dataset using Python. It demonstrates data analysis and visualization using Pandas and Matplotlib.

## Tools Used
- Python
- Pandas
- Matplotlib
- CSV

## Dataset
The dataset contains:
- Order ID
- Order Date
- Product
- Category
- Region
- Quantity
- Price
- Total Sales

## Features
- Read CSV data
- Analyze sales data
- Category-wise sales analysis
- Bar chart visualization
- Line chart visualization
- Pie chart visualization

## Files
- `sales_analysis.py` – Python analysis script
- `sales_report.csv` – Sales dataset
- `README.md` – Project documentation

## Output
This project generates charts to visualize sales performance by category and over time.
